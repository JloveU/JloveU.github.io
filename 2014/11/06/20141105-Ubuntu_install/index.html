本文为在已安装Windows7系统的前提下安装Ubuntu Kylin 14.10系统的过程以及期间出现的各种问题的记录。

Ubuntu系统下载

Ubuntu Kylin中文官方网站：http://www.ubuntu-china.cn/

Ubuntu Kylin 14.10 64位下载地址：
http://cdimage.ubuntu.com/ubuntukylin/releases/14.10/release/ubuntukylin-14.10-desktop-amd64.iso

Ubuntu Kylin 14.10 32位下载地址：
http://cdimage.ubuntu.com/ubuntukylin/releases/14.10/release/ubuntukylin-14.10-desktop-i386.iso

将系统镜像刻录到U盘

我选择安装的是64位版系统，即 ubuntukylin-14.10-desktop-amd64.iso。

用 UltraISO 软件将此系统镜像刻录到U盘。

给Ubuntu分配空间

从硬盘中分出一定大小的分区，我使用的是Windows7自带的卷压缩功能，将最后一个盘压缩出了20G空间。（有人说最好把分配给Ubuntu的分区放在硬盘的最后一段，为的是避免原Windows系统分区识别出错。）

将此分区删除，使之处于空闲状态即可。

重启电脑，开始安装

插入刚才制作的启动U盘，重启电脑，从U盘启动。

2. 分配空间

安装过程中要为系统中的不同目录分配空间。一般要为以下4项分配空间：

  • /boot：Linux系统内核所在目录
  • swap：虚拟内存
  • /：Linux系统根目录
  • /home：用户数据根目录

/boot：Linux系统内核只有几十M，分配200M左右即可。

swap：虚拟内存，如果电脑配置的内存小于2G，则需要swap为实际内存的2倍，否则swap和实际内存大小相同即可，比如我的电脑装了4G内存，那么我为swap分配了4G空间。

/：这个是Linux系统的根目录，包括安装的软件等，可以分配大一点，我分配了15G。

/home：分配完上面3项之后，将剩下的2G左右空间分配给了这部分。因为Linux系统可以访问Windows的NTFS硬盘，所以这部分可以不用分配太大。

3. 添加启动项

Ubuntu安装完之后重启，但是此时还不能进入Ubuntu系统，因为没有指向Ubuntu的启动项。

进入Windows7系统，运行EasyBCD软件，点击“添加新条目”->“Linux/BSD”，输入启动项名称，“驱动器”选择对应的 /boot 所在分区，点击“添加条目”。

重启。即可看到对应Ubuntu的启动项。

新安装的Ubuntu系统的优化配置

安装完Ubuntu系统后，又遇到了一些问题：

1. 鼠标闪烁

首次进入Ubuntu系统时，可能会出现鼠标闪烁的问题，鼠标经过的地方会留下痕迹。

解决办法是：

进入系统设置，点击“显示”，会发现有两个显示器，选中“未知显示器”，设置为“关闭”，确定退出即可。

参考链接：
如何解决 Ubuntu 系统 鼠标 指针跳动的问题_百度经验

2. 时间快了8小时

Ubuntu系统安装的时候选择了所在时区，但是安装完之后时间比北京时间快了8小时。

进入系统设置，修改一下时区，或者手动修改一下时间即可。
